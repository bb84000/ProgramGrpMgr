English description at the end of french description

GESTIONNAIRE DE GROUPE DE PROGRAMMES

OBJET
Création de groupes de programmes dans une fenêtre sur le bureau.

INSTALLATION
Exécuter le programme d'installation contenu dans l'archive, l'installateur choisit 
la version 32bit ou 64 bit selon l'OS.

UTILISATION
Pour créer un groupe de programmes à partir de raccourcis et/ou de fichiers contenus
dans un répertoire, cliquer sur le bouton de gauche dans la barre d'outils et
sélectionnez le répertoire désiré.

Pour ajouter un programme à partir d'un raccourci ou d'un fichier exécutable, cliquer
sur le bouton du milieu et sélectionnez le fichier désiré.

Pour sauvegarder le groupe de programme ainsi créé, quittez le programme, la boîte
de dialogue qui s'affiche vous permet de choisir le nom du groupe de programme,
de placer un raccourci sur le bureau et, en double cliquant sur l'icone, de choisir
l'icon de ce raccourci.

Pour exécuter un programme, double cliquer sur l'icone de ce programme ou utilisez
le menu contextuel (clic droit de la souris) qui permet d'exécuter le programmme en
mode administrateur. Ce menu permet également de supprimer l'icone d'un programme du groupe.

DESINSTALLATION
Cliquez sur le lien de désinstallation dans le menu Démarrer ou utilisez le système
de manipulation des programmes dans le panneau de configuration.

AVERTISSEMENT
Utilisez ProgramGrpMgr à vos risques et périls. L'effacement d'une icone dans le
gestionnaire de groupe de programmes ne supprime pas le raccourci ni le programme.

NOTICE
Le logiciel ProgramGrpMgr peut être utilisé gratuitement pour une utilisation non commerciale.

French description at the beginning of this document

PROGRAM GROUP MANAGER

OBJECT
Creates programs groups in a window on the desktop.

INSTALLATION
Run the setup program in the archive, the installer uses the proper 32 bit or 64 bit version corresponding to your OS.


USAGE
To create a programs group from shortcuts and/or files containend in a folder,
click on the left button in the tools bar and select the proper folder.

To add a program from a shortcut or an executable file, click on the middle button
in the tools bar and select the proper file.

To save the program group created, quit the program, the dialog box displayed
let you choose the name of the program group, place a shortcut on the desktop,
and, with a double click on the cion, select an icon for this shortcut.

To run a program, double click on its icon or use the context menu (mouse right clic
on the program icon, whic allows to run the progream in administator mode. The menu
also allows to delete a program icon in the group.

UNINSTALL
Click on the uninstall link in the Start menu or use the program management system
in the control panel.

INFORMATION
Use ProgramGrpMgr at your own risks. Deleting an icon in the Programs group manager
don't delete the sortcut or the program.
